## Attestation Infrastructure on Stellar

Modular attestation protocol on Stellar Soroban with BLS delegation and payment-gated verification.

## Key Features

  <Card title="BLS Delegation" icon="key">

  <Card title="Economic Security" icon="shield-check">

## Quick Start

<CodeGroup>
```typescript Initialize
import { StellarAttestationClient } from '@attestprotocol/stellar-sdk';

const client = new StellarAttestationClient({
  rpcUrl: 'https://soroban-testnet.stellar.org',
  network: 'testnet',
  publicKey: 'GBXXXX...'
});
```

```typescript Attest
await client.attest({
  schemaUid: Buffer.from('abc...', 'hex'),
  value: JSON.stringify({ verified: true }),
  subject: 'GSUBJECT...',
  options: { signer }
});
```
</CodeGroup>
